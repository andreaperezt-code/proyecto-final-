 Mascotas Perdidas

Aplicación web desarrollada en Python con Flask que permite reportar, visualizar y dar seguimiento a animales perdidos. El proyecto nace a partir de un problema real: la dificultad de encontrar información clara y actualizada cuando una mascota se pierde.

Este proyecto fue desarrollado como prototipo funcional aplicando el proceso de Design Thinking y el enfoque de Vibe Coding.

Objetivo del proyecto

Crear una herramienta simple y accesible que permita centralizar la información de animales perdidos, facilitando la colaboración de las personas y evitando que los reportes se pierdan en redes sociales.

Usuario objetivo

* Personas que han perdido a su mascota
* Personas que desean ayudar a encontrar animales perdidos
* Voluntarios y comunidad en general

Funcionalidades principales

* Página de inicio con información del proyecto
* Formulario para reportar animales perdidos
* Lista de animales reportados
* Vista de detalle de cada animal
* Actualización del estado del animal (Perdido, Visto, Encontrado)

Instrucciones de ejecución

1. Clonar el repositorio
2. Abrir el proyecto en Visual Studio Code
3. Activar el entorno virtual (si aplica)
4. Instalar Flask:

   pip install flask
   
5. Ejecutar la aplicación:

   python app.py

6. Abrir el navegador en:

   http://127.0.0.1:5000

Enfoque de Vibe Coding

El desarrollo del proyecto siguió un enfoque de Vibe Coding, utilizando herramientas de inteligencia artificial como apoyo para:

* Estructurar el proyecto
* Diseñar la experiencia de usuario
* Optimizar y comprender el código
